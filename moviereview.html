Movie One-Liners, March 2020

Movie One-Liners, is a full-stack application allowing registered
           users to leave a one-line comment/review of any film. Other 
           users can then also add their own comments to each movie's Hub. 
            The back-end was built using Node, Postman, PostgreSQL/Sequelize, 
            Express, Postico, SQL, and then React for the front-end.
            (Also JavaScript, HTML, CSS, API, React, Node, Express, SQL)

Not deployed/Github
